Building
---------
The application is built with sbt. See http://www.scala-sbt.org/ for more details.

Tests can be run with the command:
sbt test

Running
--------
The application can be run using the command:
sbt run

The urls are then entered on newlines, followed by -1 once all input has been completed.
The application will then print out the size of each webpage downloaded.
